Updated last July 15, 2014

We hope you will find this template helpful in getting your project started. We've built in some common features that you may need. Here's a quick overview:

Schema.org tags: Schema tags help Google know more about your project when it turns up in a search. By default, this blueprint inherits from the NewsArticle schema tag. We define a headline, author, publish date, article body, and comments. You can change this by overriding the schema block on the body tag. New to schema tags? Not sure if NewsArticle is right for your project? Read our handy documentation to get started.

Ads: Our ad library is present in Tarbell projects by default. You can read more about how to use it here, including how to use the standard ad map, or just crib off of our examples on this page.

Bootstrap 3: We include our custom version of Bootstrap 3 in this template. Bootstrap 3 introduced different names for grid components. Of note, Bootstrap requires a containing element to wrap site contents and house our grid system. You may choose one of two containers to use in your projects. Note that, due to padding and more, neither container is nestable. Use .container for a responsive fixed width container. Use .container-fluid for a full width container, spanning the entire width of your viewport. Read more here before you start building!
